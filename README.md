## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Technologies used

- Next.js 13
- MongoDB
- TailwindCSS
- Clerk authentication
- ParticleJS
- Typewriter
- Animate.css
- Gemini Pro API
